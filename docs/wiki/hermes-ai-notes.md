# Hermes AI Notes

Hermes can refer to different AI projects. For this repo, use the term carefully.

## Useful Meanings

| Meaning | Use in this project |
| --- | --- |
| Hermes model family | A possible lower-cost or local model for summarizing, classifying, and drafting notes. |
| Hermes-style agent | A long-running automation agent that can learn repeatable workflows and turn them into reusable skill files. |

## Where Hermes Can Help

Hermes-style automation is useful for repeatable support work around the archive:

- Draft a first-pass Skill summary from notes.
- Suggest scenario tags and platform tags.
- Compare a new Skill against the evidence-first review checklist.
- Draft 3 good points, 3 can-be-better points, and use cases for human review.
- Turn a successful workflow into a draft wiki page.
- Create a weekly queue of Skills to test next.

## Guardrails

Start narrow:

- Read-only repo access first.
- No secrets.
- No deployment tokens.
- No automatic GitHub writes.
- No moving or deleting files without explicit approval.
- Separate branch for generated changes.

Treat Hermes as a junior operator that earns access by producing useful, reviewable work.
